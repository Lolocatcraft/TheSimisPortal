Passo a passo

Para começar abra o painel de cheats do The Simis 4
Plataformas:

1. Computador: CTRL + SHIFT + C

2. Playstation 4/5: R1 + L1 + R2 + R2

3. Xbox ONE/X/S: LB + RB + LT + RT

Passo 2

Agora para vc poder usar cheats que podem mudar a vida do seus sims
precisa que vc ative o testingcheats no console
então na aba dos cheats digite:

testingcheats true

Passo 3

Agora para liberar os obigetos do debug mode / Cheats que mudam a vida dos Sims ou Familia/Grupo
digite esses dois comandos no cosole e precione Enter

bb.showhiddenobjects

bb.showliveeditobjects
